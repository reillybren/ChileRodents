Ectoparasite Sharing and Phylogenetic Relatedness

Purpose: Calculate the probability for rodent species to share ectoparasites based on phylogenetic relatedness.

Load packages and read in files. The phylogenetic distances were calculated in Geneious

library(dplyr)
library(geotax)
library(reshape2)
library(ggplot2)

Clean the genetic distance matrix, replacing NAs with zeros and accession numbers with scientific names. Export as a csv file.

dist2[is.na(dist2)]<-0 #replace NAs with 0
dist<- dist2 %>%
  rename_with(~ seq$Rodent, seq$ID)#rename columns and rows with rodent names
row.names(dist)<-colnames(dist)

Use the GeoTax package to build a binary interaction matrix between rodents and ectoparasites and perform a logistic regression on the entire dataset and for each ectoparasite.

incidence <- get_incidence_matrix(genpairs, returnDataFrame=FALSE) ## binary interaction matrix
incidence<- incidence[ ,colnames(dist)]
incidence<- subset(incidence, rowSums(incidence)>2) # get rid of rodents with less than 3 ectoparasite associations

coef_all <- log_reg_boostrap(incidence, dist, 1000) # log reg coefs for all ectoparasites
coef <- sapply(1:nrow(incidence), function(x)
  log_reg_boostrap(incidence[x, ,drop = F], dist, 1000) ) #for each ectoparasite
colnames(coef)<- rownames(incidence) # change the v1,2,3 to the ectoparasite names

The overall logistic regression coefficients for phylogenetic distance are slope: -9.9885745 and intercept: 0.480412

Now, use the logistic regression coefficients to calculate probability vectors.

GD<- seq(0, 0.4,0.001 )## scale for genetic distance
coefval <- t(coef)[ ,c(1,7)] #subset the coefs to just slope/intercept
coef_allval<- c(coef_all[["intercept"]], coef_all[["slope"]] ) #subset the coefs to just slope/intercept
reg <- apply(coefval, 1, function(x) {
  prob_logit(x, GD)} ) # vector of probabilities for each ectoparasite
reg_all <- prob_logit(coef_allval, GD) # get vector of probabilities for the entire dataset

.

The probability vectors can be used to plot the probability of sharing ectoparasites based on genetic distance between rodents.First, the probability vectors need to be combined with the scale for genetic distance (GD) and then we can plot with ggplot.

##reformat data to plot
plotreg<- cbind(reg, GD)
plotreg<-cbind(plotreg, reg_all)
plotreg<-as.data.frame(plotreg)
preg<-  melt(plotreg,  id.vars = 'GD', variable.name = 'series')

# format to get the reg all like to be red
adj_names <- sort(setdiff(unique(preg$series), "reg_all"))
gg_color_hue <- function(n) {
  hues = seq(0, 0, length = n)
  hcl(h = hues, l =80, c = 0)[1:n]
}
values <- gg_color_hue(length(adj_names))
names(values)<- adj_names
values<- c(values, c(reg_all="red"))

# plot 
ggplot(preg, aes(x=GD, y=value, colour=series))+
  geom_line(show.legend=F)+ theme_minimal()+
  scale_colour_manual(values=values)+
  xlab("Genetic Distance between Rodents")+
  ylab("Probability of Sharing Ectoparasites")
